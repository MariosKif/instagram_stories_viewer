## Hogyan szamold ki a TikTok elkotelezettsegi ratat

Tobb keplet is hasznalatos az iparban. A helyes valasztas a celjaidtol fugg.

### 1. keplet: Elkotelezettsegi rata megtekintsek alapjan (ERV)

Ez a TikTokon leggyakrabban hasznalt keplet, mert a platform terjesztesi modellje azt jelenti, hogy a megtekintsek messze meghaladhatjak a kovetoszamot.

**ERV = ((Kedvelesek + Hozzaszolasok + Megosztasok + Mentesek) / Osszes megtekintes) x 100**

Pelda: Egy video 12 000 megtekinteset, 1400 kedvelest, 85 hozzaszolast, 220 megosztast es 95 mentest kap.

ERV = ((1400 + 85 + 220 + 95) / 12 000) x 100 = **15,0%**

### 2. keplet: Elkotelezettsegi rata kovetok alapjan (ERF)

Ezt a kepletet a markak reszesitik elonyben keszitok osszehasonlitasakor, mert egy stabil szamhoz (kovetok) normalizal egy valtozo helyett (megtekintsek).

**ERF = ((Kedvelesek + Hozzaszolasok + Megosztasok + Mentesek) / Osszes koveto) x 100**

Pelda: Ugyanazok az interakciok, mint fent, de a fioknak 8000 kovetoje van.

ERF = ((1400 + 85 + 220 + 95) / 8000) x 100 = **22,5%**

Megjegyzes: az ERF a TikTokon gyakran meghaladja a 100%-ot viralis videok eseten, mivel a TikTok a tartalmat a kovetoidnel sokkal tobb embernek mutatja.

### 3. keplet: Atlagos elkotelezettsegi rata

A fiok teljes teljeasitmenyenek pontosabb kepeert szamiztsd ki az elkotelezettsegi ratat tobb friss posztbol.

**Atlagos ER = Egyedi poszt ER-ek osszege / Posztok szama**

Szamitsd ki az ERV-t vagy ERF-et az utolso 10-20 posztodra es atlagold oket. Ez kiegyenliti a viralis siketek vagy alulteljesito tartalom okozta kiugro ertekeket. ### 4. keplet: Sulyozott elkotelezettsegi rata

Nem minden elkotelezettseg egyenlo. Egy megosztas vagy mentes erosebb erdeklodest jelez, mint egy kedveles. Egyes marketingesek sulyozott kepleteket hasznalnak.

**Sulyozott ER = ((Kedvelesek x 1) + (Hozzaszolasok x 2) + (Megosztasok x 3) + (Mentesek x 3)) / Megtekintsek x 100**

A szorzok valtozhatnak preferencia szerint, de az elv az, hogy a nagyobb erofeszitesigenyuu muveletek nagyobb sulyt kapnak.

## Hogyan javitsd a TikTok elkotelezettsegi ratad
