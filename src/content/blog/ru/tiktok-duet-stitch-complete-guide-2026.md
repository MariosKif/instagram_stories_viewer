## Что такое Дуэт в TikTok

### Определение

Дуэт создаёт видео с разделённым экраном, где ваш контент воспроизводится рядом с оригиналом. По умолчанию оригинальное видео отображается справа, а ваш новый контент — слева. TikTok значительно расширил варианты компоновки Дуэта с момента запуска функции.

### Варианты компоновки Дуэта в 2026 году

| Компоновка | Описание | Лучше всего для |
|------------|---------|-----------------|
| Лево/Право | Классическая компоновка бок о бок | Реакции, комментарии, гармонии |
| Верх/Низ | Горизонтальная стопка | Танцевальные челленджи, до/после |
| Реакция | Маленькое «картинка в картинке» на фоне оригинала | Комментарии, реакции |
| Зелёный экран | Оригинальное видео как ваш фон | Добавление контекста, сторителлинг |
| Лево/Право с выравниванием | Оба видео полноразмерные, свайпом | Сравнительный контент |

### Как создать Дуэт

1. Найдите видео для дуэта (у автора должны быть разрешены дуэты)
2. Нажмите кнопку «Поделиться» (значок стрелки)
3. Выберите «Дуэт»
4. Выберите компоновку из опций внизу
5. Запишите свой контент параллельно с оригиналом
6. Отредактируйте, добавьте текст, эффекты и звуки
7. Опубликуйте с релевантными хэштегами и описанием

### Настройки конфиденциальности Дуэта

Каждый автор TikTok контролирует, могут ли другие делать дуэты с его видео:

- **Все** — любой пользователь TikTok может сделать дуэт
- **Друзья** — только взаимные подписчики
- **Выключено** — никто не может сделать дуэт

Вы можете установить настройку по умолчанию в конфиденциальности и переопределять её для отдельных видео. Если хотите, чтобы ваш контент распространялся через дуэты, всегда оставляйте настройку «Все».

## Что такое Стежок в TikTok

### Определение

Стежок позволяет вырезать до пяти секунд из видео другого автора и использовать их как начало своего. В отличие от Дуэта, где оба видео воспроизводятся одновременно, Стежок создаёт последовательный опыт — сначала воспроизводится оригинальный фрагмент, затем плавно переходит в ваш контент.

### Как создать Стежок

1. Найдите видео для стежка
2. Нажмите кнопку «Поделиться»
3. Выберите «Стежок»
4. Используйте инструмент обрезки для выбора до 5 секунд из оригинала
5. Нажмите «Далее» для начала записи ответа
6. Запишите, отредактируйте и добавьте свой контент
7. Опубликуйте с подходящими хэштегами

### Когда использовать Стежок vs Дуэт

Выбор между Стежком и Дуэтом зависит от типа создаваемого контента:

| Сценарий | Дуэт | Стежок |
|----------|------|--------|
| Реакция в реальном времени | Да | Нет |
| Добавление контекста или объяснения | Оба | Да |
| Гармонизация или танец рядом | Да | Нет |
| Ответ на вопрос | Оба | Да |
| Проверка фактов или исправление | Нет | Да |
| Создание до/после | Да | Оба |
| Продолжение чужой истории | Нет | Да |
| Сравнение бок о бок | Да | Нет |

## Креативные стратегии для Дуэтов

### Дуэт с добавленной ценностью

Самые успешные дуэты не просто реагируют — они добавляют то, чего не хватало оригинальному видео. Это может быть экспертный комментарий, альтернативная точка зрения, дополнительные факты или юмористический поворот. Аудитория вовлекается в дуэты, которые делают совместный контент лучше, чем каждое видео по отдельности.

**Примеры:**
- Шеф-повар делает дуэт с видео кулинарного лайфхака, объясняя, почему он работает (или нет)
- Юрист делает дуэт с вирусным юридическим заблуждением и разъясняет реальный закон
- Музыкант делает дуэт с певцом, добавляя гармонию или инструментальное сопровождение

### Дуэт-ответ на челлендж

Многие тренды TikTok строятся вокруг дуэтов. Танцевальные челленджи, лип-синк и креативные промпты часто распространяются через цепочки дуэтов. Участие в них, пока тренд актуален, выводит ваш контент на аудиторию, уже вовлечённую в тренд.

### Контрастный Дуэт

Расположите свой контент как прямой контраст оригиналу. Это особенно хорошо работает для лайфстайл-контента, фитнес-трансформаций и демонстрации навыков. Визуальная контрастность естественным образом удерживает внимание зрителя.

### Экспертный комментарий в Дуэте

Если у вас есть экспертиза в определённой области, дуэт с вирусным контентом в вашей нише с профессиональным анализом — один из самых быстрых способов выстроить авторитет в TikTok. Медицинские специалисты, финансовые консультанты и учёные создали огромные аудитории, используя эту стратегию.

## Креативные стратегии для Стежков

### Стежок «Вот что они не рассказали»

Вырежьте фрагмент популярного видео и предоставьте дополнительную информацию, контекст или исправления. Этот формат работает, потому что использует зацепку оригинального видео, позиционируя вас как знающего эксперта.

### Стежок — продолжение истории

Используйте фрагмент из трендового промпта (вроде «Расскажите о случае, когда...») и постройте вокруг него увлекательную личную историю. Такие видео имеют высокий процент досмотра, потому что комбинируют проверенную зацепку с оригинальным сторителлингом.

### Стежок с острым мнением

Вырежьте спорное или провокационное утверждение и поделитесь своей точкой зрения. Согласитесь, возразите или предложите нюансированную золотую середину. Этот формат генерирует высокую вовлечённость в комментариях, потому что зрители хотят поделиться собственным мнением.

### Стежок — расширение урока

Когда кто-то делится советом или лайфхаком, сделайте стежок и расширьте полноценным уроком, дополнительными шагами или альтернативными методами. Вы используете их аудиторию, предоставляя более глубокую ценность.

Да. TikTok позволяет делать дуэт с дуэтом, создавая цепной эффект. Результатом становится видео с тремя панелями (или выбранная компоновка с объединённым дуэтом как исходным материалом).
