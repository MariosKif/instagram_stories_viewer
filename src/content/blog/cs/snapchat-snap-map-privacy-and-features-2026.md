# Snap Map na Snapchatu — Funkce, nastaveni soukromi a bezpecnostni tipy pro rok 2026

> **Klicove poznatky:** Snap Map sdili vasi polohu s prateli pokazde, kdyz otevrte Snapchat. Sprava nastaveni soukromi — zejmena Ghost Mode — je nezbytna. Pouzijte [Map Viewer od IGStoryPeek](https://www.igstorypeek.com/snapchat/map-viewer) pro prozkoumani obsahu Snap Map.

## Jak Snap Map funguje

Snap Map pouziva GPS data zarizeni k umisteni avatara Bitmoji na sdilenou mapu. Poloha se aktualizuje pri kazdem otevreni aplikace. Po osmi hodinach neaktivity Bitmoji zmizi.

### Indikatory stavu Bitmoji

| Stav Bitmoji | Co indikuje |
|---|---|
| Ridi auto | Zarizeni se pohybuje rychlosti jizdy |
| Posloucha hudbu | Sluchatka pripojena |
| Na letisti | Poloha blizko letiste |
| Spi (Zzz) | Neotevreli jste aplikaci nekolik hodin |
| Na konkretnim miste | Jste na rozpoznanem miste |
