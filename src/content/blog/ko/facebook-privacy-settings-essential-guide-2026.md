수백만 사용자가 얼마나 많은 정보를 노출하고 있는지 모른 채 소셜 미디어를 탐색합니다. [IGStoryPeek](https://www.igstorypeek.com)과 같은 도구는 자신의 신원을 밝히지 않고 Instagram, Facebook, TikTok, Snapchat의 공개 콘텐츠를 볼 수 있게 도와줍니다. 첫 번째 방어선은 자신의 [Facebook 개인정보 컨트롤](https://www.igstorypeek.com/facebook)을 올바르게 구성하는 것부터 시작합니다.

이 가이드는 7가지 실행 가능한 단계로 모든 중요한 개인정보 구성을 안내합니다.

## 단계 1: Facebook 개인정보 설정 대시보드 감사

개인정보 점검 도구는 현재 개인정보 구성을 한 곳에서 검토하는 가장 빠른 방법입니다.

### 개인정보 점검 접근

설정 및 개인정보로 이동한 다음 개인정보 점검을 선택합니다. Facebook은 이를 5가지 카테고리로 구성합니다: 누가 공유 내용을 볼 수 있는지, 사람들이 당신을 찾는 방법, 데이터 설정, 광고 기본 설정, 앱 권한. 각 카테고리를 체계적으로 작업하세요.

### 대시보드가 보여주는 것

대부분의 사용자는 프로필이 생각보다 훨씬 더 많이 노출되어 있음을 발견합니다. 기본 설정은 친구의 친구가 게시물을 보고 서드파티 앱이 데이터에 액세스할 수 있도록 허용합니다.

## 단계 2: 게시물과 프로필을 볼 수 있는 사람 제어

대상 컨트롤은 어떤 그룹이 콘텐츠를 볼 수 있는지 결정합니다.

### 게시물 가시성 설정

| 설정 | 기본값 | 권장 값 |
|---|---|---|
| 향후 게시물 | 친구의 친구 | 친구 |
| 이전 게시물 | 혼합 대상 | 친구 (이전 게시물 제한 사용) |
| 프로필 정보 (생일, 도시) | 공개 | 친구 또는 나만 보기 |
| 친구 목록 | 공개 | 나만 보기 |
| 이메일 주소 | 친구의 친구 | 나만 보기 |
| 전화번호 | 친구의 친구 | 나만 보기 |

### 프로필 정보 잠금

각 프로필 정보를 개별적으로 "친구" 또는 "나만 보기"로 설정하세요. 프로필로 이동하여 세부 정보 편집을 클릭하고 각 항목의 대상을 조정하세요. 프로필 정보의 개인정보 구성은 절대 "공개"로 남겨서는 안 됩니다. [IGStoryPeek의 프라이버시 우선 브라우징 도구](https://www.igstorypeek.com)와 결합하면 디지털 존재감 주위에 완전한 보호막을 만들 수 있습니다.

## 단계 3: 검색 및 발견 Facebook 개인정보 설정 관리

이 설정은 낯선 사람이 검색 엔진과 Facebook 자체 검색을 통해 프로필을 찾을 수 있는지 여부를 제어합니다.

### 검색 엔진 색인

설정, 개인정보로 이동하여 "Facebook 외부 검색 엔진이 프로필에 링크하도록 허용하시겠습니까?"를 비활성화하세요. 활성화되면 프로필이 Google 검색 결과에 표시됩니다.

### 연락처 기반 발견

이메일과 전화번호 검색 옵션을 모두 "친구" 또는 "나만 보기"로 설정하여 데이터 유출의 연락처 정보를 통해 낯선 사람이 찾는 것을 방지하세요.

## 단계 4: 앱 권한 및 서드파티 액세스 검토

Facebook 계정에 연결된 서드파티 앱은 가장 큰 개인정보 취약점 중 하나입니다.

### 연결된 앱 감사

설정, 앱 및 웹사이트로 이동합니다. 적극적으로 사용하지 않는 모든 앱을 제거하세요. 유지하는 앱의 경우 각 앱이 보유한 정확한 권한을 검토하고 불필요한 것을 취소하세요.

### 플랫폼 API 컨트롤

| 권한 유형 | 위험 수준 | 조치 |
|---|---|---|
| 공개 프로필 액세스 | 낮음 | 신뢰할 수 있는 앱에만 허용 |
| 친구 목록 액세스 | 중간 | 모든 앱에서 취소 |
| 이메일 액세스 | 중간 | 필수 앱에만 허용 |
| 대신 게시 | 높음 | 즉시 취소 |
| 사진 액세스 | 높음 | 필수가 아닌 모든 앱에서 취소 |

서드파티 사이트에서 Facebook 로그인을 사용하지 않는 경우 "앱, 웹사이트 및 게임" 플랫폼을 완전히 비활성화하세요. 이것은 플랫폼에서 할 수 있는 가장 공격적인 개인정보 변경입니다. 소셜 미디어 브라우징에는 계정에 서드파티 뷰어 앱을 연결하는 대신 [IGStoryPeek](https://www.igstorypeek.com)을 사용하세요.

Facebook의 광고 시스템은 데이터를 사용하여 광고를 타겟팅합니다. 이 컨트롤은 광고주가 액세스할 수 있는 것을 줄입니다.

## 자주 묻는 질문

### 모든 facebook 개인정보 설정을 가장 비공개 옵션으로 재설정하려면 어떻게 하나요?

단일 "비공개로 재설정" 버튼은 없습니다. 설정 및 개인정보의 개인정보 점검 도구를 사용하여 각 카테고리를 작업하세요. 게시물 대상을 친구로 설정하고, 검색 엔진 색인을 비활성화하고, 서드파티 앱 액세스를 제거하세요.

### facebook 개인정보 설정을 조정한 후에도 사람들이 내 프로필을 볼 수 있나요?

이름과 프로필 사진은 설정에 관계없이 모든 사람에게 표시됩니다. 하지만 이러한 컨트롤을 잠그면 게시물, 친구 목록, 개인 세부 정보, 타임라인이 친구가 아닌 사람에게 숨겨집니다.

### 이 프라이버시 컨트롤이 Messenger 대화에 적용되나요?

Messenger에는 메인 Facebook 프로필과 별도의 프라이버시 컨트롤이 있습니다. Messenger 설정은 누가 연락할 수 있고 누가 활동 상태를 볼 수 있는지를 제어합니다. 포괄적으로 두 설정 세트를 모두 검토하세요.

### Facebook은 얼마나 자주 개인정보 설정을 변경하나요?

Facebook은 일반적으로 연간 2~4회 프라이버시 옵션을 업데이트합니다. 대규모 재설계 시 설정이 새로운 위치로 이동하는 경우가 있습니다. [IGStoryPeek 블로그](https://www.igstorypeek.com/blog)를 팔로우하면 모든 플랫폼의 프라이버시 변경 사항에 대한 정보를 얻을 수 있습니다.

### facebook 개인정보 설정을 변경하면 기존 게시물에 영향을 주나요?

기본 대상 설정을 변경하면 향후 게시물에만 영향을 줍니다. 이전 콘텐츠에 변경 사항을 적용하려면 설정, 개인정보의 "이전 게시물 제한"을 사용하세요. 이렇게 하면 "공개" 또는 "친구의 친구"로 설정된 모든 이전 게시물이 "친구"로 변경됩니다.

### Facebook 프로필을 완전히 보이지 않게 만들 수 있나요?

Facebook에서 완전한 비가시성은 불가능합니다. 이름과 프로필 사진은 항상 검색 가능합니다. 하지만 프라이버시 컨트롤을 최대화하면 프로필이 친구가 아닌 사람에게 기능적으로 보이지 않게 됩니다.
