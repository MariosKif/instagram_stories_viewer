## 9 viktiga tips för Facebook Marketplace integritet

Dessa nio strategier bildar ett komplett försvar för alla som handlar på denna plattform.

### Tips 1: Lås din profil först

Din Facebook-profil är den enskilt största sårbarheten. Innan du publicerar någon annons, justera dessa inställningar.

- **Ställ in din vänlista till "Bara jag."** Detta förhindrar främlingar från att bläddra bland kontakter och bygga falskt förtroende.
- **Dölj arbets- och utbildningsdetaljer.** Ta bort synligheten för anställningsinformation som bedragare använder för social manipulation.
- **Begränsa synligheten för tidigare inlägg.** Använd Facebooks "Begränsa tidigare inlägg"-funktion för att begränsa tidigare inlägg till bara vänner.
- **Granska din profil som en främling.** Använd Facebooks "Visa som"-funktion för att se exakt vad kontakter ser.

### Tips 2: Håll konversationer transaktionella

Dela aldrig ditt telefonnummer, e-post eller hemadress via Messenger. Håll dig till varan, priset och möteslogistiken. ## Vanliga frågor

### Kan någon se min adress på Facebook Marketplace?

Plattformen visar inte din exakta adress som standard. Den visar din allmänna stad eller stadsdel. Dock kan annonsfoton med ostrippade metadata avslöja GPS-koordinater. Ta alltid bort fotometadata och träffas på offentliga platser.

### Är Facebook Marketplace säkert för att sälja dyra föremål?

Att sälja högt värderade föremål medför högre risk. Använd Facebooks checkout för skickade föremål för att få betalningsskydd. För personliga försäljningar, träffas på en polisstations säkra handelszon och acceptera bara kontanter.

### Hur rapporterar jag en bedragare på Facebook Marketplace?

Öppna konversationen, tryck på trepunktsmenyn och välj "Rapportera." Du kan också trycka på "Rapportera annons" på varusidan. Spara skärmdumpar innan du rapporterar.

### Kan jag använda Marketplace utan att visa mitt riktiga namn?

Nej. Plattformen kräver ett personligt Facebook-konto med ditt riktiga namn alltid synligt. Du kan inte använda en pseudonym utan att bryta mot användarvillkoren. Detta är den mest betydande facebook marketplace integritetsbegränsningen, medvetet inbyggd av Meta för att skapa transaktionsförtroende.

### Använder Facebook mina Marketplace-data för reklam?

Facebook använder din aktivitet för att rikta annonser, inklusive vad du bläddrar bland, listar och köper. Dessa data matas in i Metas annonsprofil. Du kan begränsa annonspersonalisering genom inställningar, men fullständig avanmälan är inte tillgängligt medan du använder plattformen.

### Vad händer om jag blir lurad på Facebook Marketplace?

Om du använde Facebooks checkout kan du kvalificera dig för Köpskydd som täcker varor som inte mottagits eller felbeskrivits. För kontanttransaktioner finns inget skydd. Förebyggande genom starka facebook marketplace integritetsrutiner är mycket mer effektivt än återhämtning.
